# 📄 문서 스캐너 (android document scanner korean PWA)

> 무료 · 무제한 · 광고 없음 · 설치 불필요

스마트폰으로 문서를 촬영하고, 네 꼭짓점을 조정해서 문서만 반듯하게 잘라 PDF로 저장하는 웹 앱입니다.

---

## ✨ 주요 기능

- 📷 **카메라 촬영 / 갤러리 불러오기**
- ✂️ **수동 꼭짓점 조정** — 초록 점을 드래그해서 문서 모서리에 맞추기
- 🔲 **원근 보정** — 비스듬하게 찍어도 반듯하게 펴줌
- 🖼️ **배경 제거** — 문서 영역만 깔끔하게 잘라냄
- 📑 **PDF 저장** — 여러 장을 하나의 PDF로 묶기
- 📱 **PWA 지원** — 홈 화면에 추가하면 앱처럼 사용 가능

---

## 📱 사용 방법

### 설치 (홈 화면 추가)
1. 스마트폰 브라우저로 접속
2. 메뉴 → **홈 화면에 추가**
3. 이후 앱 아이콘으로 바로 실행

### 스캔 순서
1. **📷 촬영** 또는 **🖼️ 갤러리** 버튼 클릭
2. 초록 꼭짓점 4개를 문서 모서리에 맞게 드래그
3. **✓ 자르기** 클릭
4. 여러 장 반복 후 **📑 PDF 저장**

---

## 🚀 직접 배포하기

### GitHub Pages로 무료 호스팅

1. 이 저장소를 **Fork** 하거나 파일을 다운로드
2. 본인 GitHub 저장소에 업로드
3. **Settings → Pages → Branch: main → Save**
4. `https://[본인아이디].github.io/[저장소명]` 으로 접속

### 로컬 실행
별도 서버 없이 `index.html` 파일을 브라우저로 열면 바로 실행됩니다.

---

## 📁 파일 구조

```
/
├── index.html      ← 앱 전체 (단일 파일)
├── manifest.json   ← PWA 설정
├── sw.js           ← 서비스 워커 (오프라인 지원)
└── README.md       ← 이 파일
```

---

## 🛠️ 기술 스택

- **Vanilla JS** — 프레임워크 없음
- **Canvas API** — 이미지 처리 및 원근 보정
- **jsPDF** — PDF 생성
- **PWA** — 오프라인 지원, 홈 화면 추가

---

## 📋 촬영 팁

- 문서를 **최대한 정면**에서 촬영할수록 결과가 깔끔함
- 조명이 밝을수록 배경 제거가 잘 됨
- 꼭짓점 조정 시 문서 모서리에 정확히 맞출수록 펴짐이 좋음
- **건너뛰기** 버튼으로 꼭짓점 조정 없이 바로 저장도 가능
